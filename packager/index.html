Packager

The packager allows you to generate a .html file for a project, which can be packaged into a .exe file (somewhat) easily.

While the packager on the homepage creates a link to a website, this packager generates an easily packagable HTML file that functions independently of an internet connection.

Still in development. Note blocks will not work, some icons will be missing, etc.

Displayed while the project is loading. Set to empty to display only a loading bar. You can use HTML formatting.
For very advanced users
